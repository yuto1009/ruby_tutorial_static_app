# Ruby on Rails Tutorial's sample application

## Ruby version 
`3.1.4`
## System dependencies
```
$ gem install bundler -v 2.4.12
$ bundle _2.4.12_ install
```
## Database creation
```
$ rails db:migrate
```
## How to run the test suite
```
$ rails test
```
